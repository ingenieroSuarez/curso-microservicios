## Description

Curso de udemy por JFSS, comandos cli de nest para crear modulos:
```
nest g mo passenger
```
Crear Controller: 
```
nest g co passenger
```
Crear Servicio:
```
nest g s passenger
```

## Installation

```bash
$ npm install
```


## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
